# Component Documentation

## UI Components (`src/components/ui`)

-   **Button:** Accessible button with variants (default, outline, ghost). Supports loading state and icons.
-   **Card:** Flexible card component with header, content, and footer sections. Supports hover effects.
-   **Input/Select:** Form elements styled with Tailwind, compatible with React Hook Form.
-   **LazyMap:** Wrapper for Google Maps iframe that loads only when scrolled into view.

## Layout Components (`src/components/layout`)

-   **Navbar:** Responsive navigation bar with mobile hamburger menu.
-   **Footer:** Site footer with links and newsletter subscription.
-   **Section:** Wrapper for page sections with standardized padding and container widths. Supports scroll reveal animations.

## Home Components (`src/components/home`)

-   **Hero:** Main landing section with headline and CTA.
-   **LoanCategories:** Grid of available loan types with icons.
-   **LoanCalculator:** Interactive estimator for monthly payments.
-   **Testimonials:** Carousel of customer feedback.

## Analytics (`src/components/analytics`)

-   **WebVitals:** Client component that reports Core Web Vitals to the console (or analytics service).
